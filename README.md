# Movie Recommendation System using Machine Learning

## Project Overview

This project is a Content-Based Movie Recommendation System that suggests movies similar to a user's favorite movie based on their attributes and textual information. The system leverages Natural Language Processing (NLP) and Machine Learning techniques to identify similarities between movies and generate personalized recommendations.

## Objective

To build an intelligent recommendation engine that helps users discover movies similar to their interests using movie metadata rather than user behavior.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* TF-IDF Vectorization
* Cosine Similarity
* Difflib

## Methodology

1. Data preprocessing and handling missing values.
2. Selection of key features:

   * Genres
   * Keywords
   * Tagline
   * Cast
   * Director
3. Combining selected features into a single text representation.
4. Converting text data into numerical vectors using TF-IDF.
5. Calculating movie similarity using Cosine Similarity.
6. Matching user input with the closest movie title and generating recommendations.

## Key Skills Demonstrated

* Natural Language Processing
* Recommendation Systems
* Similarity Analysis
* Machine Learning Workflow

## Outcome

The system successfully recommends movies with similar characteristics, demonstrating the practical application of NLP and content-based recommendation techniques in real-world entertainment platforms.
